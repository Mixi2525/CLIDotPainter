# CLIDotPainter

`CLIDotPainter`は、ターミナル上で動作するシンプルなドット絵お絵描きツールです。レイヤー機能や図形描画機能などの機能があります。

## 主な機能

-   **レイヤーベースの編集**: 複数のレイヤーを作成・管理し、様々な描画が可能です。サイドパネルには現在選択中のレイヤーが `>` マークで表示されます。
-   **キャンバス**: カーソルを操作してキャンバス上を移動し、描画を行うことができます。
-   **情報の表示**: レイヤー一覧やカラーパレットを表示するためのサイドパネルがあります。

## 依存関係

このプロジェクトをビルド・実行するには、以下の環境が必要です。

-   C++コンパイラ (`g++` や `clang++` など)
-   `make`
-   `ncurses` ライブラリ (開発用ファイル。Debian/Ubuntuでは `libncurses-dev`, Fedoraでは `ncurses-devel` など)

## ビルド方法

1.  **リポジトリをクローンします (まだの場合):**
    ```bash
    git clone <your-repository-url>
    cd CLIDotPainter
    ```

2.  **`make`コマンドを実行してプロジェクトをコンパイルします:**
    ```bash
    make
    ```
    実行ファイルは `bin/` ディレクトリに生成されます。

## 実行方法

ビルドが成功したら、以下のコマンドでアプリケーションを起動できます。

```bash
./bin/main
```

## 操作方法

-   カーソル移動: `hjkl` または 矢印キー
-   描画: `space`
-   プログラムの終了: `q`
